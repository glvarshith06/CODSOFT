## Task Details

### Task 1 — Chatbot with Rule-Based Responses
**File:** `task1_chatbot.py`

A simple rule-based chatbot that responds to user inputs using regex pattern matching.

**Features:**
- 15+ predefined rules covering greetings, farewells, jokes, help, and more
- Pattern matching using Python's `re` module
- Clean conversational loop with exit support

**How to Run:**
```bash
python task1_chatbot.py
```

---

### Task 2 — Tic-Tac-Toe AI
**File:** `task2_tictactoe_ai.py`

An unbeatable Tic-Tac-Toe AI agent using the Minimax algorithm with Alpha-Beta Pruning.

**Features:**
- AI plays optimally and never loses
- Alpha-Beta Pruning for faster decision making
- Human vs AI terminal-based gameplay
- Option to choose who goes first

**How to Run:**
```bash
python task2_tictactoe_ai.py
```

---

### Task 4 — Recommendation System
**File:** `task4_recommendation_system.py`

A movie recommendation system using both Collaborative and Content-Based Filtering techniques.

**Features:**
- **Content-Based Filtering** — TF-IDF + Cosine Similarity on movie genres
- **Collaborative Filtering** — User-based cosine similarity on ratings
- **Hybrid Recommendation** — Combines both methods for better accuracy
- Interactive CLI to try your own inputs

**Dependencies:**
```bash
pip install pandas numpy scikit-learn
```

**How to Run:**
```bash
python task4_recommendation_system.py
```
